Technical Symbols

While building the OpenPPL library the developers had to create lots of internal supporting functions. Most of them are so technical that they are of no use for the end-user. However some of them might be helpful for people who want to extent the OpenPPL library with their own symbols. A function like SmallBlindChair might for example be useful for the development of PokerTracker symbols like PT_SmallBlind_VPIP.

Symbol

Remarks

Chair0StartingStackSize ... Chair9StartingStackSize

Starting stacksize of Chair N at the beginning of the hand (balance + currentbet)

Preflop only

ConstCardTwo ... ConstCardAce

Named card constants to improve readability and maintenability. Useful if you want to access OpenHoldems card symbols $$pr0, $$pr1, $$cr0 ... $$cr4.

None

CommonCard0Paired ... CommonCard4Paired

True, if the rank of common card N is equal to the rank of another common card.

Postflop only

RankOfSetOnBoard

Rank of the set on board

Valid only, if there are 3 or 4 cards of the same rank, undefined otherwise

EarlyPosition1Chair ... MiddlePosition3Chair, CutOffChair, ButtonChair, SmallBlindChair, BigBlindChair

Chair numbers of the specific players. Could be used e.g. for symbols like PT_SmallBlind_VPIP

None
