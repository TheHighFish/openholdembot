PokerTracker Symbols

OpenHoldem supports PokerTracker symbols, and so does OpenPPL. The development team did already create some symbols of general use that get shipped with the library. However: the possibilities are endless and you can for sure imagine other symbols that are useful. If you want to increase the power of OpenPPL please have a look at the chapter “Building Symbols On Your Own” and at Openholdems manual. It is not that difficult and if you want to contribute to the project your symbols could even become part of the next version.

Note: The following symbols need a working connection to Poker Tracker Database (PT2/PT3 depinding on OH version), correct opponents name scraping and sitename/network vars in the TableMap.

Number of hands symbols are needed to weight reliability of the symbol you are interested in. Infact for some symbol 80 hand could perfectly represent an Opponent playing style while for other symbols you could be in the need of hundred or thousand hands. Minumum hands symbols can be changed editing the Library to fit own needs.

Please note that OH returns percentage values divided by 100; So for example 50% is returned as 0.50.
