0.1. Poker Tracker symbols

Assuming all the prerequisites as described above are met, the following symbols will be available to your bot for use in its logic processing. The appropriate use of these symbols in opponent modeling is beyond the scope of this document, but plenty of references can be found on the Internet with a Google search.

Version 3 of PokerTracker introduced new stats; some of them are now supported by OH. Be careful to not use PT3-stats with an installation of PT2!

0.1.1. Ring symbols (PT2 + PT3)

Meaning

pt_iconx (x=0-9)

Poker Tracker auto-rate icon code for chair x

Poker Tracker flop aggression for chair x

Poker Tracker saw flop pct for chair x

Poker Tracker saw turn pct for chair x

pt_riverpctx (x=0-9)

Poker Tracker saw river pct for chair x

pt_vpipx (x=0-9)
