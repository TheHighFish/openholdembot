Custom Formula

From Maxin Wiki

Jump to: navigation, search

It's a function that you can create using Openholdem symbols and other custom formulae to represent a situation. His name has the "f$" suffix, same as the standard formulae, and you can name it what you want (ex. f$big_stack_folded).
