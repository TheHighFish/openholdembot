BettingAction Symbols

Name Meaning Limitations / Remarks
AllOpponentsLeftSittingOut True, if all remaining opponents are sitting out. This symbol is especially meant for use in tournaments None
Bets The number of bets made this betting round, can only be 0 or 1 since the 2nd bet is a raise
Contrary to Standard PPL Bets is always 1 preflop.
Only valid at my turn with autoplayer engaged (see “Raises”).
BigBlindSittingOut True, if the big blind is sitting out. This symbol is especially meant for use in tournaments None
BotCalledBeforeFlop True if we called preflop None
BotCalledOnFlop True if we called on the flop Postflop only
BotCalledOnTurn True if we called on the turn Turn and River only
BotCalledOnRiver True if we called on the river River only
BotIsLastRaiser We have the betting / raising initiative None
BotRaisedBeforeFlop We raised Preflop, can also be used on the Preflop and will be true if we bet or raised None
BotRaisedOnFlop We bet or raised on the Flop, can also be used on the Flop and will be true if we bet or raised Postflop only
BotRaisedOnTurn We bet or raised on the Turn, can also be used on the turn and will be true if we bet or raised Turn and River only
BotsActionsOnFlop Number of actions where we put money in the pot on the Flop None
All-ins are not counted obviously (as the game would be over anyway)
BotsActionsOnThisRound Number of actions this betting round where we put money in the pot None
All-ins are not counted obviously (as the game would be over anyway)
BotsActionsOnThisRoundIncludingChecks Number of checks + number of actions where we put money in the pot None
All-ins are not counted obviously (as then the game would be over anyway)
BotsActionsPreflop Number of actions where we put money in the pot Preflop, all-ins are not counted None
BotCheckedPreflop True if bot checked Preflop None
BotCheckedOnFlop True if bot checked on Flop None
BotCheckedOnTurn True if bot checked on Turn None
BotCheckedOnRiver True if bot checked on River None
BotsLastAction Bot’s last action, can be one of the following: none, beep, raise, bet, call, or check None
BotsLastPreflopAction Bot’s last action Preflop, can be one of the following: none, beep, raise, bet, call, or check None
Calls The number of calls by opponents on this betting round Only valid at my turn with autoplayer engaged (see “Raises”).
CallsSinceLastRaise The number of calls by all opponents since the last raise by an opponent on the current betting round None
Checks The number of checks made by opponents this betting round Only valid at my turn with autoplayer engaged (see “Raises”).
Folds The number of folds this betting round None
MissingSmallBlind True, if there is no small blind in this hand, e.g. the player who would have been SB did bust in the previous hand. None
NoBettingOnFlop No bets/raises were made on the Flop, may also be used on the Flop and will be true if there were no bets/raises Postflop only
NoBettingOnTurn No bets/raises were made on the Turn, may also be used on the Turn and will be true if there were no bets/raises None
NoVillainBetOrRaisedOnFlop No villan bet or raised on Flop None
NoVillainBetOrRaisedOnTurn No villan bet or raised on Turn None
NumberOfOpponentsAllin Number of opponents who raised or called allin. Range: 0..9. If the amount to call is equal to our stack size and we are headsUp we consider the villan as being allin.
NumberOfRaisesBeforeFlop The number of raises before the Flop made by opponents. None
NumberOfRaisesOnFlop The number of raises on the Flop made by opponents. Bets don’t count. Postflop only
NumberOfRaisesOnTurn The number of raises on the Turn made by opponents. Bets don’t count. Turn and river only
OpponentCalledOnFlop An opponent called on Flop, and did not raise or bet Postflop only
OpponentCalledOnTurn An opponent called on Turn, and did not raise or bet Turn and river only
OpponentIsAllin An opponent is all in, there may still be other players in the hand If the amount to call is equal to our stack size and we are headsUp we consider the villan as being allin.
Raises The number of raises made by opponents this betting round. Bets don’t count. Only valid at my turn with autoplayer engaged. The calculation depends on game-history (multiple frames) and needs to be done exactly once per orbit, very shortly before the autoplayer acts. The symbol is undefined before and becomes undefined immediatelly thereafter again. For other times when it is not the players turn (f$prefold) use “easy” symbols without that restriction like AmountToCall.
RaisesBeforeFlop The number of raises by opponents before flop None
Calculation depends on game-history (multiple frames)
RaisesOnFlop The number of raises by opponents on flop. Bets don’t count. Postflop only
Calculation depends on game-history (multiple frames)
RaisesOnTurn The number of raises by opponentson turn. Bets don’t count. Turn and River only
Calculation depends on game-history (multiple frames)
RaisesSinceLastPlay The number of raises since our last action. Bets don’t count. None
SmallBlindSittingOut True, if the big small is sitting out. This symbol is especially meant for use in tournaments None
Please note: some of these symbols depend on game history and on counters that get updated once per orbit, i.e when it is our turn to act. They are well-defined only when it is our turn and undefined otherwise.
