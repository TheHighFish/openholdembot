OpenPPL-Mode

OpenHoldem will switch the formula editor into OpenPPL mode as soon as the formula contains a non-empty f$preflop function. Then all primary autoplayer-functions will be hidden and a subtree with the four main OpenPPL functions will be shown instead (f$preflop, f$flop, f$turn and f$river).
If your formula-editor is in OH-script mode then just create a new user-defined function, call it “f$preflop”, start editing it, apply your changes and reopen the editor.
